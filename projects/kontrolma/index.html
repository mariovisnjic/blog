About

My brother approached me with a simple request: "I need a simple website that displays the services I provide, as cheaply as possible, ideally for free."

This is the third rendition of this website. The first one was React app because I was learning React at the time. The Second was WordPress because I was going for simplicity. After he forgot to pay for hosting for the WordPress site and the site got deleted, I had the freedom of choice to restart in a new way.

The website is not the primary source of leads for his company, so I have a lot of freedom.

Stack

I'm obsessed with keeping it simple recently, so my first reaction was HTML + CSS and nothing else. Remembered that Bootstrap 5 has template examples of full pages which can be used for free. Proud to say that I did not add a line of JavaScript in it. Since it is a static page, I used Netlify to host it. The first version of the new website was up and running in 15 minutes. The Croatian government provides a free domain for your business as long it is a similar name to a registered company.

Given template needed a few changes and was ready to run:

  • I reused all the copy from before
  • Changed images from .png to .webp
  • Defer image loads
  • Change Bootstrap and Bootstrap icons to be imported from cdn
  • Small fixes based on lighthouse suggestions
  • Automate Netlify forms with Captcha to work and send email correctly

Lighthouse score

In the end, all was done in few hours purely because of me wanting to optimize it as much as possible.
